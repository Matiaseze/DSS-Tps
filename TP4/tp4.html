# Normalizar los datos

# PCA

# t-SNE

e. Qu ventajas tiene aplicar la reduccin de dimensionalidad sobre nuestro dataset?

Las principales ventajas de aplicar reduccin de dimensionalidad sobre el dataset son:

  • Mejora el rendimiento de los algoritmos: Menos variables hacen que los modelos sean ms rpidos y eficientes.
  • Reduce el riesgo de sobreajuste: Elimina ruido y variables irrelevantes, ayudando a que el modelo generalice mejor.
  • Facilita la visualizacin: Permite representar datos complejos en 2D o 3D para anlisis y presentacin.
  • Simplifica el anlisis: Hace ms fcil interpretar los resultados y encontrar patrones relevantes.
  • Disminuye el almacenamiento y el costo computacional: Menos datos ocupan menos espacio y requieren menos recursos para procesar.
